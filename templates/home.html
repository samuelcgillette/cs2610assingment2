This is a homework assignment for my cs2610 class. This is my first http webserver. It will send back the appropriate responses based on the request. It is connected to a database that contains the information for the movies, seasons, and tv pages. The home page is just a welcome page with links to the other pages. The movies page will show a list of movies, the seasons page will show a list of seasons, and the tv page will show a list of tv shows. Each page has its own css file that styles the page. The home page also has a js file that logs a message to the console when it is loaded.
